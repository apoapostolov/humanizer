# Humanizer Skill

Humanizer is a writing-editing skill for removing common signs of AI-generated prose while preserving meaning, audience fit, and voice.

Version: `1.0.0`

- `SKILL.md` - the skill definition
- `EXAMPLES.md` - concrete before-and-after examples for example-focused models
